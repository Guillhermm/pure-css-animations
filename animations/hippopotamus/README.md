# Hippopotamus

A playful, animated hippo illustration created entirely with CSS.

This animation demonstrates how CSS can be used to build and animate complex shapes and characters without any JavaScript.

![Hippopotamus preview](./preview.png)

## Features

- **Character Design**: Full hippo made with divs and CSS shapes.
- **Subtle Animations**: Breathing effect, ear wiggle, and blinking eyes add life to the character.
- **Interactive Details**: Hovering over the nose and teeth triggers small animations for extra fun.
- **Pure CSS**: No JavaScript required â€” everything is handled with HTML + CSS.

## Running Locally

1. Clone this repository:

```bash
git clone https://github.com/Guillhermm/pure-css-animations.git
```

2. Navigate to the animation folder:

```bash
cd animations/hippopotamus
```

3. Open the `index.html` file in any modern browser.

## Notes

- The animation scales well for typical desktop and mobile screens.
- Best viewed on modern browsers supporting CSS animations and transforms.
