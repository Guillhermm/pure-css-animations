# Dynamic Gradient

A full-screen, CSS-only animated gradient background with a centered text overlay, creating a smooth, colorful motion effect.

![Dynamic Gradient preview](./preview.png)

## Features

- **Animated Gradient**: Smoothly transitions through multiple colors using `@keyframes`.
- **CSS-Only Implementation**: No JavaScript; all animation and text overlay handled purely with CSS.
- **Responsive Text Overlay**: Centered message scales for different screen sizes.
- **Full-Screen Background**: Covers the entire viewport with continuous gradient motion.
- **Lightweight & Modern**: Minimal CSS, ideal for hero sections, splash screens, or visual effects.

## Running Locally

1. Clone this repository:

```bash
git clone https://github.com/Guillhermm/pure-css-animations.git
```

2. Navigate to the animation folder:

```bash
cd animations/dynamic-gradient
```

3. Open the `index.html` file in any modern browser.

## Notes

- The gradient moves infinitely with a 15-second animation loop.
- Overlay text uses the **Poppins** font and has a subtle text shadow for readability.
- Fully compatible with modern browsers supporting CSS `linear-gradient` and `@keyframes`.
